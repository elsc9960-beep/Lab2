## Тема: Розробка функціональності Web-додатка мовою Javascript 

### Завдання роботи:
Розробити функціональність для статичних сторінок Web-додатку першої лабораторної роботи із використанням шаблону MVC. 

**Обрана тематика:** Калькулятор (режим звичайний, для програміста).

### Особливості реалізації:
- **Архітектура:** Проєкт організовано у вигляді модулів (класів) за шаблоном MVC (Model-View-Controller).
- **Інструменти:** HTML5, CSS3, Javascript. Web-фреймворки (ReactJS, AngularJS, VueJS) не використовувалися згідно з вимогами.
- **Зберігання даних:** Для збереження даних користувачів (облікові записи, поточна сесія) використано `localStorage` та базові структури даних Javascript.

### Функціональні можливості додатка:
- **Звичайний калькулятор:** Виконання базових арифметичних операцій.
- **Калькулятор програміста:** Підтримка систем числення (HEX, DEC, OCT, BIN) та побітових логічних операцій (AND, OR, XOR, NOT, зсуви).
- **Система користувачів:** Повноцінно реалізовано реєстрацію нових користувачів та вхід до сайту.
- **Профіль:** Можливість перегляду інформації профілю та редагування особистих даних.
